Sequelize
  - Object-Relational Mapping library.
  - Sequelize is a promise-based ORM for Node.js v4 and upwards. It supports 
    the dialects PostgreSQL, MySQL, SQLite and MSSQL and features solid 
    transaction support, relations, read replication and more.
  - An ORM is known as Object Relational Mapper. This is a tool or a level of 
    abstraction which maps(converts) data in a relational database into 
    programmatic objects that can be manipulated by a programmer using a 
    programming language(usually an OOP language).

    {js Object} =====Mapped===> SQL row

  - Sequelize offerce:
    * Models.
    * Instances these Models.
    * Queries
    * Assocition
